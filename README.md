# 崔老师课题组年终总结展示系统

一个仿 Google Gemini 风格的年终总结展示网页应用，专为课题组年终汇报设计。

## 📋 项目概述

这是一个静态网页项目，模拟 Gemini AI 对话界面，用于展示课题组的年终总结。用户可以通过输入框或点击快捷卡片触发年终总结幻灯片演示。

## 🌟 主要功能

### 1. Gemini 风格主界面
- 深色渐变主题（深紫色调）
- 顶部显示"崔老师小组年终总结"标题
- 中央欢迎区域包含老师头像（带渐变发光边框）和标题"欢迎来到崔老师的课题小组"
- 仿 Gemini 的输入框和快捷提示卡片

### 2. 聊天对话功能
- 支持用户输入问题，模拟 AI 回复
- 流式打字输出效果（逐字显示）
- 预设关键词回复规则（在 `js/app.js` 的 `chatResponses` 中配置）

### 3. 年终总结幻灯片演示
- 点击快捷卡片或输入相关关键词触发
- 支持多种幻灯片类型：开场页、统计数据、照片展示、文字卡片、结尾页
- 数字滚动动画效果
- 键盘左右箭头或点击屏幕导航

### 4. 春节主题特效
- 雪花飘落动画（❄ ❅ ❆ 等符号）
- 节日元素随机飘落（🏮 🧨 🎊 🧧 ✨ 🎆）
- 特效同时显示在主界面和年终展示界面

## ⚙️ 自定义配置

### 修改幻灯片内容
编辑 `js/config.js` 文件：

```javascript
const CONFIG = {
    teamName: "XX课题组",     // 团队名称
    year: "2025",             // 年份
    loadingDuration: 1500,    // 加载动画时长(毫秒)
    autoPlayInterval: 0,      // 自动播放间隔(0=禁用)
    
    slides: [
        // 幻灯片数组，支持以下类型：
        // intro - 开场页
        // stat  - 统计数据（带数字滚动动画）
        // photo - 照片展示
        // text  - 纯文字卡片
        // outro - 结尾页
    ]
};
```

### 修改对话回复
编辑 `js/app.js` 中的 `chatResponses` 对象来自定义 AI 回复规则。

### 修改特效
编辑 `js/effects.js` 中的 `config` 对象来调整雪花数量、节日元素等。

## 🎨 设计特点

- **深色渐变主题**：`#0f0c29 → #1a1a3e → #24243e`
- **强调色渐变**：蓝紫粉渐变 `#667eea → #764ba2 → #f093fb`
- **Google Fonts**：使用 Noto Sans SC 字体
- **玻璃拟态效果**：半透明背景 + backdrop-filter 模糊

## 🚀 使用方法

1. 直接用浏览器打开 `index.html` 即可运行
2. 或者使用本地服务器（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve .
   ```

## 💡 常见修改

| 需求 | 修改位置 |
|------|----------|
| 更改团队名称 | `js/config.js` → `teamName` |
| 添加/修改幻灯片 | `js/config.js` → `slides` 数组 |
| 更换照片 | 替换 `images/` 目录下的图片 |
| 修改 AI 回复 | `js/app.js` → `chatResponses` |
| 调整雪花数量 | `js/effects.js` → `config.snowflakeCount` |
| 修改主题颜色 | `css/style.css` → `:root` 变量 |
